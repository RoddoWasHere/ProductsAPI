Setup

Configure database

To configure your local database settings, navigate to the file 'adapter/database_adapter'. In the 'DatabaseAdapter' class, provide the appropriate host, username and password. Once you have saved the file, reload this page and click 'Initialize Database'(below). This will create a database called 'ProductsRoderichDB'. When you're done with the application, you can drop the database by clicking 'Drop Database'(below).

Once the database has been successfully initialized, you can then navigate to the home page Here

Note: Reinitializing the database will clear any data previously stored.
